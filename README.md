# 🌐 Website Responsivo - Modelo Base
## 📖 Sobre o Projeto

### Este é um projeto de website responsivo desenvolvido com HTML, CSS e JavaScript, ideal para servir como base para:

Sites institucionais
Landing pages
Portfólios pessoais
Apresentação de negócios

### O foco principal é oferecer uma interface moderna, simples e adaptável, garantindo uma boa experiência para o usuário em diferentes dispositivos.

🚀 Funcionalidades
✅ Menu de navegação responsivo
✅ Layout moderno com seção Hero
✅ Botões de ação (Call To Action)
✅ Design adaptável para mobile
✅ Menu mobile com abertura e fechamento via JavaScript
## 🛠️ Tecnologias Utilizadas
HTML5 → Estrutura do site
CSS3 → Estilização e responsividade
JavaScript (Vanilla) → Interatividade
Font Awesome → Ícones
